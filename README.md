*** Dino Crisis Early Cutscene Skip mod ***

** ABOUT **

LUA Script de Cheat engine que detecta cutscenes y las skippea. Se intenta hacer una solución más elegante que simplemente usar el speedhack para acelerar las cutscenes, lo que impacta tambien en el IGT. Con esto, se puede terminar el juego en menos de una hora.

** HOW TO USE **

Cargar el juego en Cheat Engine, y luego cargar el archivo DINO.CT para asociar el LUA con el juego. Al aceptar la ejecución del script, ya estará activado el cutscene skip.

Combinar con [DinoReady Patch DoorSkip](https://www.speedrun.com/dino1/resources/kzgcz) para una experiencia muchísimo más fluida.
